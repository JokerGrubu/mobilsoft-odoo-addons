Sequence Dynamic Placeholders
==================================

ENGLISH
----------

Overview:
----------
The Sequence Dynamic Placeholders module is designed to enhance the functionality of sequences in Odoo by using dynamic placeholders. This module allows users to define placeholders for month and year, and replace these placeholders with custom characters or values based on specific mapping configurations.

Features:
----------
- Dynamic Placeholders: Easily configure month and year placeholders in sequences.
- Custom Mapping: Define custom characters or values for placeholders for each sequence.
- User-Friendly Interface: Easy setup and configuration options.

Installation:
----------
- Download the module files and place them in your Odoo addons directory.
- Update the app list in Odoo.
- Search for "Sequence Dynamic Placeholders" in the Apps menu.
- Click the install button.

Usage:
----------
After installation, you can access dynamic placeholder configuration options in the sequence settings. You can define how the month and year placeholders behave and what values they represent.

Sekans İçin Dinamik Yer Tutucu
==================================

Genel Bakış: Sequence Dynamic Placeholders modülü, Odoo'daki sekansların işlevselliğini dinamik yer tutucular kullanarak geliştirmek için tasarlanmıştır. Bu modül, kullanıcıların ay ve yıl için yer tutucular tanımlamasına ve bu yer tutucuların belirli eşleme yapılandırmalarına göre özel karakterler veya değerlerle değiştirilmesine olanak tanır.

Özellikler:
----------
- Dinamik Yer Tutucular: Sekanslarda ay ve yıl yer tutucularını kolayca yapılandırın.
- Özel Eşleme: Her bir sekans için yer tutuculara özel karakterler veya değerler tanımlayın.
- Kullanıcı Dostu Arayüz: Kolay kurulum ve yapılandırma seçenekleri.

Kurulum:
----------
- Modül dosyalarını indirip Odoo eklenti dizininize yerleştirin.
- Odoo'da uygulama listesini güncelleyin.
- Uygulamalar menüsünde "Sequence Dynamic Placeholders"ı arayın.
- Kurulum butonuna tıklayın.
- Kullanım: Kurulumdan sonra, sekans ayarlarında dinamik yer tutucu yapılandırma seçeneklerine erişebilirsiniz. Ay ve yıl yer tutucularının nasıl davranacağını ve hangi değerleri temsil edeceğini tanımlayabilirsiniz.
