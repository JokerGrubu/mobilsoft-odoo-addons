# JOKER Bank Integration Module

Turkish Banks Open Banking API Integration for Odoo 19

## Features

- **Garanti BBVA** - Complete OAuth 2.0 integration
- **Ziraat Bank** - Corporate and retail account support
- **Multi-Currency** - Automatic exchange rate updates
- **Transaction Sync** - Automated transaction fetching
- **Reconciliation** - Smart transaction matching

## Installation

1. Copy module to Odoo addons directory
2. Update apps list
3. Install "JOKER Bank Integration"

## Configuration

1. Go to **Bank Integration > Configuration > Garanti BBVA**
2. Create new connector
3. Enter OAuth credentials (from bank's developer portal)
4. Click "Connect"
5. Configure auto-sync settings

## Requirements

- Odoo 19.0
- Python requests library
- Bank API credentials (sandbox or production)
